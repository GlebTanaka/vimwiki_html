What is Java

  • Short answer, Java is a programming language. To be precise, it's an object oriented language. It has also some functional programming in it.
  • Objected oriented means four things:
    1. Abstraction
    2. Encapsulation
    3. Inheritance
    4. Polymorphism
  • it does not support multiple inheritance, meaning one class can only extend one other class
  • Java is a statically typed language:
    • any type of variable you create is known at compiler time. Even if you define a type of var, the compiler will guess the type of this variable.

What is the Object class in Java

  • the super class of every class you can create.
  • when you create a class, you can explicitly extend another class with the extend keyword.
    • even the Object class
    • by default the compiler makes you extend the super class.
    • that means every class in Java extends the Object class.
