Sichbarkeits- und Gültigkeitsbereiche

Der Gültigkeitsbereich eines Namens ist der Bereich eines Programmes, wo durch den Namen auf die Variable zugegriffen werden kann.

  • Method Scope
    • Wird eine Varibale innerhalb einer Methode deklariert, so besitzt sie den Method Scope. 
          int doSomething(String info) {
        int length = info.length();
        return length;
    }
  • Block Scope
    • Variablen die innerhalb von Blöcken deklariert werden, sind nur in den Zeile zwischen den curly brackets zugreifbar
    • startend ab der Deklaration 
int first = 1;
{
    int second = 2;
    {
        int third = 3;
    }
}
  • Class Scope
    • Variablen in einer Klasse werten als Attribute bezeichnet.
    • Sie sind fuer alle Methoden der Klasse sichbar.
    • Bei Namenskonflikten mit Methoden-Parametern kann man diese mit this direkt ansprechen.
